# Typing Adventure Kids

Lightweight React typing games website for kids.

## Tech
- React + Vite
- No backend
- No auth
- No database
- localStorage only

## Features
- Home page with instant guest play
- 10 typing games with score, speed, accuracy, replay
- Timed typing test with optional player name
- Result card screen with PNG download (`html-to-image`)
- Local progress dashboard with best scores and recent result
- About page for parents and teachers

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
